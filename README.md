# artmeta

This code helps package security and privacy research artifacts for
evaluation. The code asks a series of questions from an artifact author
and produces metadata.toml file encapsulating the answers. It
also enforces presence of some required answers and their specific format.
One can use CTRL-C to abort, and save partial answers. When the
code runs again it will retrieve past answers and the user can
keep or modify them, then proceed with the remainder of the
questions.

Any questions or comments should be sent to mirkovic@isi.edu.
If you have bug fixes please submit a pull request.
